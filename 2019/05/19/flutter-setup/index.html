MacOS系统要求

  1. 操作系统:macOS(64位)
  2. 磁盘空间:700 MB(不包括IDE/工具的磁盘空间)。

flutter常用的命令

命令 描述
flutter 列出所有的命令
flutter create 创建一个新的flutter项目
flutter doctor 显示有关安装工具的信息
flutter doctor -v 查看有关安装工具详细信息
flutter run 在附加设备上运行flutter应用程序
flutter analyze 分析项目的Dart代码
flutter attach 附加到正在运行的应用程序
flutter bash-completion 输出命令行shell完成设置脚本
flutter build 构建命令
flutter channel 列出或切换分支
flutter clean 删除build/和.dart_tool/目录
flutter config 配置设置
flutter devices 列出所有连接的设备
flutter drive 为当前项目运行Flutter驱动程序测试。
flutter emulators 列出、启动和创建模拟器
flutter format 格式化一个或多个dart文件。
flutter help 显示Flutter的帮助信息。
flutter install 在附加设备上安装一个Flutter应用程序。
flutter logs 显示运行Flutter应用程序的日志输出。
flutter make-host-app-editable 将主机应用程序从生成的目录移动到非生成目录，以便编辑它们
flutter packages 用于管理Flutter包
flutter precache 填充Flutter工具的二进制工件缓存
flutter screenshot 从一个连接的设备上截屏
flutter test 为当前项目运行Flutter单元测试
flutter upgrade 升级Flutter
flutter version Flutter版本列表
